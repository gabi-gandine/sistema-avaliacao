Perfis do Sistema

Perfis definem as permissões e acessos dos usuários no sistema. Cada usuário deve ter exatamente um perfil associado.

Nome do Perfil Descrição Nível de Acesso Usuários Ativos Permissões
PERFIL Descrição do perfil Máximo Alto Médio Básico 0
Permissões do perfil PERFIL
  • Gerenciar usuários (criar, editar, ativar, desativar)
  • Gerenciar perfis e permissões
  • Visualizar logs de auditoria completos
  • Exportar dados do sistema
  • Acesso total a todos os módulos
  • Gerenciar cursos e unidades curriculares
  • Criar e gerenciar turmas
  • Criar e gerenciar processos avaliativos
  • Vincular formulários a processos
  • Visualizar relatórios consolidados
  • Exportar relatórios por turma/curso
  • Criar e editar formulários de avaliação
  • Gerenciar questões dos formulários
  • Visualizar turmas em que leciona
  • Visualizar relatórios de seus formulários
  • Exportar respostas dos formulários
  • Visualizar formulários disponíveis
  • Responder formulários ativos
  • Editar respostas (quando permitido)
  • Visualizar suas turmas
  • Visualizar histórico de respostas (quando não anônimo)
Nenhum perfil cadastrado no sistema. O sistema requer pelo menos os 4 perfis básicos: ADMINISTRADOR, COORDENADOR, PROFESSOR e ALUNO.
Nota: Os perfis do sistema são fixos e não podem ser removidos. As permissões são definidas no código da aplicação e controladas via Spring Security.
